新手建议：

1. 先用浏览器打开 [chatgpt.com](https://chatgpt.com/)；

3. 不要从不明站点下载“破解版”“中文特别版”或来路不明的安装包。

### 先确认所在地和支付方式是否受支持

ChatGPT 并未在所有国家和地区开放。注册、登录或订阅前，应先确认所在地和支付方式符合 OpenAI 当前支持范围。从不受支持的地区访问，或向不受支持地区提供访问，可能导致账号被限制或暂停。具体以 [OpenAI 支持地区说明](https://help.openai.com/en/articles/9131992-chatgpt-and-api-services-in-unsupported-countries-and-regions) 为准。

## 第二步：注册自己的账号

目标只有一个：拥有可长期登录的独立账号，而不是来路不明的共享号。

常见注册路径如下：

1. 打开 ChatGPT，选择注册 / Sign up；
2. 使用当前页面支持的邮箱或登录方式；
3. 按页面完成邮箱或其他登录验证；
4. 登录后能正常发消息，即可开始使用。

详细步骤与材料准备见：[ChatGPT 怎么注册](https://www.jrjrz.com/blog/chatgpt-zenme-zhuce/)。

### ChatGPT 和 Codex 的手机验证要分开看

对 ChatGPT 网页版和一般账号创建来说，创建新的 OpenAI 账号或使用 ChatGPT 不再要求手机验证。[查看官方说明](https://help.openai.com/en/articles/8983040-what-does-phone-verification-look-like)

但如果你还要使用 Codex，手机验证码目前仍是用户常见的核心卡点之一；在登录或首次使用流程中，账号可能直接遇到验证码页面。遇到这种情况：

* 先确认验证码页面确实来自 OpenAI 或官方登录流程；
* 优先使用自己长期可控、能够继续接收短信的境外号码；
* 没有合适号码时，再比较实体卡、eSIM、临时接码和长效接码服务；
* 验证码只填在官方页面，不要把账号密码、Cookie、Session 或其他登录凭证交给接码方；
* 如果收不到短信，先检查号码、短信拦截和页面是否支持其他验证方式。

手机验证通常只能通过短信完成；部分国家和地区可能提供 WhatsApp 选项，不能改用邮件或电话口述验证码。[OpenAI 验证说明](https://help.openai.com/en/articles/8983027)

### 没有境外手机号，可以怎么选？

| 方案            | 适合谁                      | 主要注意事项                     |
| ------------- | ------------------------ | -------------------------- |
| 自己办理实体卡或 eSIM | 准备长期使用 Codex，希望号码持续由自己维护 | 需要办卡、激活和持续保号               |
| 临时接码          | 非主力账号，只想尝试完成一次验证         | 号码通常不能长期保留，二次验证时可能无法找回     |
| 长效接码服务        | 不想自己办卡，但希望在约定服务期内还能接收短信  | 购买前确认服务期限、号码地区、接码次数和失败处理规则 |

如果你准备长期使用 Codex，或者担心后续再次验证，不建议只看一次性接码的低价格。露梦提供可在约定 **1–3 个月服务期内**接收短信的国外长效号码方案，适合暂时没有可用境外手机号、又不想自己维护实体卡或 eSIM 的用户。

> **Codex 卡在手机号验证？** 先看 [ChatGPT / Codex 手机号验证教程与长效接码方案](https://www.jrjrz.com/blog/chatgpt-shoujihao-yanzheng/)。文中对比了 giffgaff、Saily、临时接码和长效接码的差别，并提供对应服务入口。具体号码地区、服务期限、库存和交付规则以购买页面实时说明为准。

## 第三步：免费版先会用的五件事

注册成功后，先别急着付费。用免费版把下面五件事跑通，才知道自己到底缺不缺 Plus。

### 1. 用完整句子说清任务

与其输入“帮我写点东西”，不如这样说：

> 我是运营，请把下面产品卖点改成 3 条朋友圈文案，每条不超过 80 字，语气轻松。

### 2. 让它按结构输出

例如要求“先给结论，再给步骤，最后列风险”。结构化要求通常比只说“写详细一点”更容易得到可用结果。

### 3. 多轮修改，而不是一次赌运气

第一版不满意时，直接说：“更短一点”“更正式”“保留第 2 点，重写第 3 点”。

### 4. 需要时再上传文件

合同摘要、表格清洗、代码报错截图等任务，把材料放进对话通常比纯口述更高效。文件上传、数据分析和具体额度会随账号、套餐和当前功能开放情况变化；如果提示额度或功能受限，记下来，这是评估升级的信号之一。

### 5. 把好用的对话固定成习惯

对重复任务（周报、翻译口径、代码审查清单），保存自己的提问模板，比收藏很多别人的提示词更有用。

### 新手使用原则

* **它会说错**：重要事实、数字和引用要复核；
* **不要直接上传敏感资料**：密码、验证码、身份证件、未脱敏客户资料等应先删除或脱敏；
* **检查数据控制设置**：可以在 ChatGPT 的 Data Controls 中管理“Improve the model for everyone”等选项，参考[官方说明](https://help.openai.com/en/articles/7730893-chatgpt-data-controls)；
* **免费版可能限流**：高峰或高频时可能被打断，见[额度用完怎么办](https://www.jrjrz.com/blog/chatgpt-edu-yongwan-zenme-ban/)。

## 第四步：什么时候再考虑 Plus

可以从任务而不是“新鲜感”出发判断：

1. 每周多次使用，并经常做到一半被额度限制打断；
2. 需要更稳定地处理文件、长文、代码或连续研究任务；
3. Free 的功能或额度已经明显影响交付速度；
4. 你算得清省下的时间大于一个月的订阅成本。

官方 Plus 标准价为 **20 美元/月**，按月计费；实际扣款可能受税费、汇率和购买渠道影响。Plus 仍可能有随系统状态变化的使用上限，不等于无限使用。

* 买共享号或合租号处理工作文件；

## ChatGPT 新手常见问题

### 新手必须先下载 App 吗？

不必。网页版就能开始，App 适合手机场景，不是上手前提。

### ChatGPT 注册必须用手机验证码吗？

对 ChatGPT 网页版和一般新账号创建来说，目前不是普遍要求。OpenAI 官方说明已明确这一点。但 Codex 的部分登录或首次使用流程仍可能要求手机验证码，两者不要混为一谈。

### Codex 为什么还会要求手机验证码？

Codex 的登录或首次使用流程可能触发额外验证，具体以账号当时显示的页面为准。如果没有可用的境外号码，可以比较自持实体卡、eSIM、临时接码和长效接码方案；准备长期使用的主力账号，应优先考虑后续仍能接收短信的号码。

### 长效接码和一次性接码有什么区别？

一次性接码通常只解决当前这一条短信，号码之后可能无法继续使用；长效接码是在约定服务期内继续保留接收短信的能力，更适合担心 Codex 二次验证的用户。它不等于号码永久归属，也不能保证平台以后一定不会调整验证规则。具体可查看[手机号验证教程与长效接码说明](https://www.jrjrz.com/blog/chatgpt-shoujihao-yanzheng/)。

### 不会英文界面能用吗？

可以。你直接用中文提问即可；界面语言随版本和设置变化，不必为此安装所谓“中文破解版”。

### 免费版能用多久？

免费版不是“试用三天到期”，而是长期提供基础能力，但额度、模型和工具权限可能有限，并会随产品规则变化。

### 一定要开 Plus 吗？

不一定。轻度使用 Free 往往够用；当额度、文件处理或连续任务已经影响工作效率时，再评估 Plus。

### 电脑和手机是同一个账号吗？

同一个 OpenAI 账号可以在多个设备上登录，订阅也应通过购买 Plus 的同一账号访问；移动端订阅还会受 Apple 或 Google 账户关联影响。以当前账号状态和官方说明为准。

### 只差付款这一步怎么办？

已有自己的账号、但官方支付反复失败时，可以先看支付排查文章；如果不想维护外卡、礼品卡或虚拟卡，再查看 [ChatGPT 开通协助方案](https://www.jrjrz.com/chatgpt)。
